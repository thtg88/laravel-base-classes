## Table of Contents

* [Usage](#usage)
* [License](#license)

## Installation

``` bash
composer require thtg88/laravel-base-classes
```

You can publish the configuration file and views by running:
```bash
php artisan vendor:publish --provider="Thtg88\LaravelBaseClasses\LaravelBaseClassesServiceProvider"
```
